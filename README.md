# Georges-Moussa-Projects
This repository showcases my projects developed from May 2024 to the present. Each folder is structured into two sub-folders: Documentation and Code.

1- Documentation: Contains comprehensive project reports and engineering diagrams such as ER diagrams and Data Flow Diagrams (DFDs).

2- Code: Houses all the files and code used for each project.

Each project created as part of a university course will be placed in a folder named after the corresponding course in addition to the project's name. Extracurricular projects will be numbered.

Notably, all these projects (except CSCI490) were developed for real local businesses. I conducted interviews with business owners to gather information for each system, subsequently developing and hosting these systems online for their use.
